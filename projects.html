My Projects

Project 1: Arduino Connect 4

Designed and programmed an LED connect 4 game on a 7x7 grid of 5V individually addressable LEDS put in cardboard.
Controlled by a breadboard with buttons connected to an Arduino Uno running a script I made, available on my GitHub.

Project 2: PCB LED Heart

Made in collaboration with WPI's IEEE, I designed a circuit in KiCad and had it cut out by PCBWay
Once it arrived I soldered the components on and then flashed it and programmed a light pattern with Arduino Uno

Project 3: Wordle Unlimited

Created the classic NYT game "Wordle" in Java to be run locally. Based roughly off a Stanford demonstration,
I scraped the dictionaries from the NYT and added a "Hint" button, along with a "Reset" for unlimited play,
a "Reveal" button for giving up, custom messages based on how well you do, and some other little tricks.
